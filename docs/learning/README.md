# AI Research Assistant: Learning Guide

Welcome to the comprehensive, phase-by-phase learning guide for the AI Research Assistant project. This directory contains detailed, concept-by-concept explanations perfectly aligned with the project's architecture.

## How to Use This Guide

This guide is designed to be read sequentially. Each phase builds upon the last. Whenever a concept is introduced, we show you **exactly where and how it is used in this project's source code**.

### Curriculum Map

1. **[Phase 0: Prerequisites](./00_Phase_0_Prerequisites.md)**
   - Git, HTML/CSS/JS basics, Python async/types, REST APIs.
2. **[Phase 1: Full-Stack Foundation](./01_Phase_1_Full_Stack_Foundation.md)**
   - Clean Architecture, FastAPI, Pydantic, React 19, TypeScript, Pytest.
3. **[Phase 2: Cloud AI Integration](./02_Phase_2_Cloud_AI_Integration.md)**
   - AWS Bedrock, boto3, Streaming Responses.
4. **[Phase 3: LiteLLM & Multi-Tenancy](./03_Phase_3_LiteLLM_Multi_Tenancy.md)**
   - API Gateway pattern, LiteLLM proxy, Token tracking, Workspaces.
5. **[Phase 4: AI Agent](./04_Phase_4_AI_Agent.md)**
   - DSPy ReAct framework, Tools (Wikipedia, Memory), Assistant-UI.
6. **[Phase 5: Temporal Workflows](./05_Phase_5_Temporal_Workflows.md)**
   - Async distributed execution, Activities, Workers.
7. **[Phase 6: MLOps & Optimization](./06_Phase_6_MLOps_Optimization.md)**
   - DSPy GRPO, MLflow tracking, Model Registry.

Let's begin the journey!
